💡 使用提示

  • 所有数据都是前端模拟的，刷新页面后部分操作会重置
  • 用户登录状态保存在 LocalStorage 中
  • 建议使用 Chrome 或 Firefox 浏览器以获得最佳体验
  • 支持移动端访问，可以在手机浏览器中打开
  • Markdown 编辑器支持实时预览和常用格式
  • 管理员账号可以访问管理后台的所有功能

🛠️ 技术栈

  • HTML5 - 语义化标签和现代 HTML 特性
  • CSS3 - Grid、Flexbox、CSS Variables、动画
  • JavaScript - 原生 ES6+，无任何框架依赖
  • LocalStorage - 本地数据存储
